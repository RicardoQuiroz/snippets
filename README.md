# Pedazos (Snippets) de código HTML
Para ser reutilizados en proyectos con HTML

## Youtube
Snippet que permite recortar o reproducir segmentos específicos de videos de YouTube.
Permite definir con precisión el segundo o fotograma de inicio y de fin de la reproducción. Se debe envolver el iframe en un contenedor <div> responsivo
https://ricardoquiroz.github.io/snippets/youtube.html

## Youtube incrustado en Genially
1. Crear una etiqueta con TinyMCE expandido en Genially
2. incrustar el código del HTML de abajo
###NOTA:
El código de inscrustación no se puede insertar directamente (YT rechaza el iframe porque detecta que está siendo usado en aplicaciones de terceros).
Por eso el iframe del player de YT debe ser insertado en otro iframe para engañar a YT y lograr que el player funcione.
https://ricardoquiroz.github.io/snippets/genially-yt.html
